## The people data object

```json
{
  "customFields": [
     "<customFields object>"
  ],
  "description": "",
  "emails": [
    "<email object>"
  ],
  "firstName": "Matt",
  "fullName": "Matt Brown",
  "jobRoles": [
    "<jobRole object>"
  ],
  "lastName": "Brown",
  "locations": [
    "<location object>"
  ],
  "notes": [
    "<note object>"
  ],
  "owner": {
    "<user object>"
  },
  "phoneNumbers": [
    "<phoneNumber object>"
  ],
  "prefix": "Mrs",
  "profilePhotoUrl": "https://linkedin.com/matt-profile-pic.png",
  "socialMedia": [
    "<socialMedia object>"
  ],  
  "tags": [
    "<tag object>"
  ],
  "Timezone": "London-GMT+0"  
}
```

### id
Unique `string` identifying the object.  This property is always set.
