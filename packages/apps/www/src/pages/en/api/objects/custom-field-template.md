## The customFieldsTemplate data object

```json
{
  "length": 0,
  "mandatory": true,  
  "max": 0,
  "metadata": {
    "<metadata object>"
  },
  "min": 0, 
  "order": 0,
  "templateName": "my custom template",
  "templateType": "TEXT"  
}
```

### id
Unique `string` identifying the object.  This property is always set.
